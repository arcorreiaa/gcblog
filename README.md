## Installing / Getting started

```bash

# Clone o repositório
$ git clone https://github.com/arcorreiaa/gcblog.git

# Entre na pasta
$ cd gcblog

# Instale as dependencias
$ yarn install

# Inicie o site
$ yarn start

```
